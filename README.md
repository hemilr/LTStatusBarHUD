# LTStatusBarHUD
简单易用的状态栏指示器框架

### 显示成功信息
```objc
[LTStstusBarHUD showSuccess:@"加载成功"];
```

### 显示失败信息
```objc
 [LTStstusBarHUD showError:@"加载失败"];
```

### 显示加载信息
```objc
[LTStstusBarHUD showLoading:@"正在加载..."];
```


### 隐藏HUD
```objc
[LTStstusBarHUD hide];
```

### 显示文字信息
```objc
[LTStstusBarHUD showMessage:@"HAHAHA"];
```
